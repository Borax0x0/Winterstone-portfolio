## Quick Start

```bash
# Clone & install
git clone https://github.com/Borax0x0/Winterstone.git
cd Winterstone
npm install

# Create .env.local with your keys
MONGODB_URI=your_db_uri
NEXTAUTH_SECRET=your_secret
NEXTAUTH_URL=http://localhost:3000

# Run
npm run dev
```
